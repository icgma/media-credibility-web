## 💡 搜索技巧

### 中英文混合搜索
数据库支持中英文双向搜索：
- "卫报" 或 "Guardian" 都能找到
- "纽约时报" 或 "New York Times" 都能找到
- "经济学人" 或 "Economist" 都能找到

### 关键词模糊匹配
- 输入"日报" → 匹配所有包含"日报"的媒体
- 输入"新闻" → 匹配所有包含"新闻"的媒体
- 输入"中国" → 匹配所有包含"中国"的媒体

### 别名支持
许多媒体有多个名称：
- CCTV = 央视 = 中央电视台
- Bloomberg = 彭博社
- The Guardian = 卫报
- SCMP = 南华早报

## 🔄 数据维护

### 自动保存
- 数据库首次加载时保存到 LocalStorage
- 后续访问直接从 LocalStorage 读取
- 无需网络连接即可使用

### 重置数据库
如需恢复到最新版本：
1. 点击"媒体库"标签
2. 点击"🔄 重置为默认"
3. 确认操作

⚠️ **注意**: 这将删除所有自定义添加的媒体

### 导出备份
建议定期导出备份：
1. 点击"媒体库"标签
2. 点击"📥 导出数据"
3. 保存 JSON 文件
