# 🧠 智能媒体数据库功能说明

## ✨ 新功能概述

现在系统内置了一个智能媒体知识库，支持**模糊搜索**和**自动匹配**，用户只需输入媒体名称（甚至部分关键词），系统就能自动识别并填充所有信息！

---

## 🎯 核心功能

### 1. 智能搜索
- ✅ **模糊匹配**：输入"央视"即可匹配"央视新闻"、"中央电视台"、"CCTV"
- ✅ **实时搜索**：输入时即时显示匹配结果
- ✅ **下拉建议**：最多显示10条最相关的结果
- ✅ **一键选择**：点击搜索结果自动填充所有字段

### 2. 内嵌数据库
- ✅ **预设数据**：内置50+常见媒体信息
- ✅ **本地存储**：使用 LocalStorage 保存，无需后端
- ✅ **持久化**：关闭浏览器后数据依然保留
- ✅ **可管理**：查看、搜索、重置、导出

### 3. 自动填充
- ✅ **类型识别**：自动判断官方/主流/商业/自媒体
- ✅ **国家识别**：自动识别中国、美国、英国等
- ✅ **注册时间**：自动填入注册日期
- ✅ **锁定编辑**：自动填充后字段锁定，防止误改

## 🚀 使用方法

### 方法1: 智能搜索（推荐）

1. **输入关键词**
   ```
   在"媒体名称"输入框中输入，例如：
   - "央视" → 匹配央视新闻、中央电视台、CCTV
   - "人民" → 匹配人民日报
   - "澎湃" → 匹配澎湃新闻
   - "BBC" → 匹配BBC中文网
   ```

2. **查看搜索结果**
   - 下拉框显示匹配的媒体列表
   - 每个结果显示：名称、类型、国家、注册日期

3. **点击选择**
   - 点击任意一条结果
   - 系统自动填充所有字段
   - 字段变为只读状态（防止误改）

4. **计算评分**
   - 点击"计算评分"按钮
   - 查看详细的评分结果

### 方法2: 手动输入

如果搜索不到想要的媒体：
1. 输入媒体名称
2. 手动选择类型、国家、日期
3. 点击计算评分
4. （可选）可以添加到数据库

### 方法3: 批量导入

在"批量评分"标签页中：
1. 准备 JSON 格式的媒体列表
2. 粘贴到文本框
3. 点击"批量计算"
4. 查看统计分析和图表

## 🗄️ 数据库管理

### 查看媒体库

1. 点击"媒体库"标签页
2. 查看所有已收录的媒体
3. 使用搜索框快速查找
4. 查看类型徽章（颜色编码）

### 搜索功能

在媒体库页面：
- 按名称搜索
- 按类型搜索（如："官方媒体"）
- 按国家搜索（如："中国"）
- 实时过滤显示

### 重置数据库

如果需要恢复默认数据：
1. 点击"🔄 重置为默认"按钮
2. 确认操作
3. 数据库恢复到初始状态

⚠️ **注意**：这将删除所有自定义添加的媒体

### 导出数据

备份或分享数据库：
1. 点击"📥 导出数据"按钮
2. 自动下载 JSON 文件
3. 文件名格式：`media-database-YYYY-MM-DD.json`

## ❓ 常见问题

### Q1: 为什么搜索不到某个媒体？

**A:** 该媒体可能还未收录到数据库中。你可以：
1. 手动填写信息进行评分
2. 联系作者添加该媒体
3. （未来版本）自行添加

### Q2: 数据会丢失吗？

**A:** 数据保存在浏览器 LocalStorage 中：
- ✅ 关闭浏览器不会丢失
- ✅ 重启电脑不会丢失
- ❌ 清除浏览器数据会丢失
- ❌ 更换浏览器会丢失

**建议**：定期导出备份

### Q3: 如何添加新媒体？

**A:** 目前版本暂不支持直接添加，但你可以：
1. 手动填写评分
2. 导出数据库
3. 编辑 JSON 文件
4. （未来版本）直接在界面添加

### Q4: 可以共享数据库吗？

**A:** 可以！
1. 导出数据库为 JSON 文件
2. 分享给其他人
3. 他们可以导入使用（未来版本支持导入）

### Q5: 搜索速度慢怎么办？

**A:** 当前数据库很小（50+条），搜索非常快。即使扩展到1000条，也应该在毫秒级完成。
